# Notification system
A notification alerts a user with important information like breaking news, product features, product events, offerrings
It has become indispensable part of our daily life

Three types of notificatoin formats:
- mobile push notification
- sms message
- email


# Understand scope and establish design scope
Building a scalable system that sends out millions of notifications a day is not easy task. 

- What types of notifications does the system support?
Push, SMS, and email

- Is it real time?
Soft real time system. As soon as possible. 

- What are the supported devices?
iOS, android, laptop

- What triggers notifications?
Notifications can be triggered by client applications

- How many notifications are sent out each day?
10 millions mobile push, 1 million SMS, 5 million emails
